# Source：iWiki 文档树总结（4013196232）

## For Agent

对 iWiki 页面 `4013196232` 下 22+ 篇子文档的总结，是一套 **AOE3D UI 基建文档索引**。覆盖四大方向：通用 UI 组件、统一交互/弹窗体系（见 [[unified-ui-interaction]]）、性能与工程化、渲染/适配/疑难问题（见 [[ui-rendering-pitfalls]]）。核心规范：能统一就统一、能数据驱动就不硬编码、重活下沉 C#、UI 异常先查渲染链路。

## 关键事实

### 通用 UI 组件 / 编辑器工具
- UI 控件库规范：基础控件做成模板 prefab，Excel 批量生成或编辑器窗口放置。
- lua 常用容器组件：[[ListBox]]/UIListView/[[UIListViewEasy]]、[[TemplateBox]]/UITemplateBox、裸写三类。
- ListView 分页拉取：`SplitPageController/Data/View`，按 `startIdx/endIdx` 拉取，解决超大网络包。
- 筛选框、多 EditBox 校验（`UIFormatView`）、Text 打字机、ListBox 聚拢动画。

### 提示 / 弹窗 / 消息体系（→ [[unified-ui-interaction]]）
- Tips 查询文档：`ShowTips1~4`、`ShowConfirmDialog`、`ShowYes/YesNo/...` 总索引。
- 系统消息 Format 规则：`DisplayPara` 各类型转展示文本。
- MsgBox：统一用新版 `ShowPopupMsgBox` 体系。
- 提示条统一入口；物品通用 Tips；通用改名弹窗；功能说明弹窗；触发式弹窗 `ShowStackUIOnSceneEmpty`；`MultiInStack` Flag（栈中多实例）。

### 时间 / 性能
- 基于 C# 的倒计时（替代 Lua 每秒 setText）。
- 指定时间戳回调机制（服务端时间戳替代本地 scheduler）。

### 渲染 / 适配 / 疑难（→ [[ui-rendering-pitfalls]]）
- 3D Canvas 透明异常（Mask 改 RT Alpha）；UGUI 动态 mask FadeTexture（顶点 fadeIndex 未刷新）；动态分辨率超边界（坐标空间混用）；Android 多窗口 16:9 letterbox 适配；Editor 加载 UI 失败。

## 推荐阅读顺序（原文）

控件库 → lua 容器组件 → Tips → 消息 Format → MsgBox → C# 倒计时 → 时间戳回调 → 校验/筛选/分页 → 渲染适配 → MultiInStack。
