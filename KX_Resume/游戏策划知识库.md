我日常使用的 AI 工具主要包括：Claude Code（代码生成与调试）、VS Code Copilot（IDE 内联补全）、ComfyUI（图像生成与原型视觉素材）以及Claude（方案讨论与文档撰写）。其中 Claude Code已深度融入我的开发工作流，以下以《卡皮巴拉 vs. 老奶》项目中"堆叠（Stacking）"玩法的开发为例说明。

- 背景： 项目要求两名玩家可以叠在一起移动，叠加状态下解锁团队专属技能，是整个游戏策略深度的核心机制。开发周期只有3周。
  创意到技术构想： 设计阶段我与策划讨论后确定了堆叠的触发条件（玩家重叠区域 +  同时按键）、物理表现（上方角色跟随下方角色刚体运动）以及输入合并逻辑（两个 Input Action 合并为一个虚拟控制器）。技术难点在于：Unity 物理系统默认不支持角色控制器之间的父子约束，需要手动同步 Transform。

- 使用 Claude Code 编写基础脚本： 我将上述技术构想以自然语言描述给 Claude Code，让它生成 StackController.cs 的骨架，包括触发检测、Attach/Detach 状态机、以及 FixedUpdate 中的位置同步逻辑。Claude Code 生成的初版脚本节省了约60% 的样板代码编写时间，让我可以直接聚焦在核心逻辑的验证上。

- 测试与 Edge Case 分析： 初版跑通后，玩家测试中暴露出几个典型 bug：
  Bug 1：堆叠时碰撞体重叠导致物理爆炸,两个 CharacterController 同时启用时相互排斥。我将报错信息和脚本片段喂给
  Claude Code，它准确定位是碰撞层未屏蔽，给出了禁用上方玩家碰撞体并改用触发器的方案。
  Bug 2：快速连续触发 Attach/Detach导致状态错乱,在极短时间内反复进出堆叠区域会使状态机卡在中间态。我描述了复现路径，Claude Code协助我分析出根因是状态切换缺少冷却锁，并帮助我补充了 Coroutine 防抖逻辑。

整个流程中，AI 扮演的角色不是替代思考，而是加速从"我知道要做什么"到"跑通第一个可测试版本"的距离，以及在debug中，帮我结构化地描述问题、缩小排查范围。

### 问题 5 游戏品类分析

#### 一、SPG 体育游戏

**SPG 体育游戏品类分析 -- 以《NBA 2K》系列 &《全明星街球派对》为例**

**1. 核心体验对比：两种不同的"篮球幻想"**

我对篮球游戏的理解，是从 NBA 2K 开始的。2K 的核心设计哲学是"拟真"——它想让你相信你就在球场上。投篮有节奏条，运球有动量惯性，防守有脚步判定，每一个细节都在模拟真实篮球的物理逻辑。玩 MyCareer 的时候，我能清楚感受到自己的球员在成长：从一个菜鸟到能在 NBA 稳定出场，那种进步是可见的、是有重量感的。这种设计满足的是一种"我真的在打篮球"的幻想。

但这套逻辑放到手游上几乎不可能成立。手机屏幕、触控操作、碎片化时间——这三个条件决定了你不能让玩家去适应高学习曲线。街球派对的选择是聪明的：它放弃了拟真，转而追求"爽感"。3v3街球本身就自带一种松散的、有个性的气质，技能设计夸张、球星招牌动作显眼、进球反馈直接——它满足的不是"我在打篮球"，而是"我在用科比打篮球"的幻想。这个差异看似微小，但对用户粘性的影响是根本性的。

两款游戏各自锚定了不同的用户心理需求：2K 给的是"沉浸与掌控"，街球派对给的是"认同与表达"。前者的门槛更高，流失也更快；后者的体验更浅，但情感锚点更稳。

**2. 数值与成长系统：为什么"肝"会让人上瘾**

2K 的 MyTeam 模式和街球派对都有卡牌/球星收集机制，但设计逻辑截然不同。

2K MyTeam 的成长是"积累型"的：你需要通过大量比赛赚取 MT 币，或者氪金开包，慢慢构建一支有竞争力的阵容。问题在于赛季结束时阵容清零，这对玩家的积累感是巨大的损耗，很多人肝了一赛季最后选择离开。它的留存逻辑依赖的是赛季限定奖励制造的短期焦虑，而不是长期成长认同。

街球派对的设计更懂手游玩家的心理。球星不是简单的数值容器，每个球星都有专属的招牌技能和视觉表现，抽到一个喜欢的球星本身就是一种"身份认同"的满足——不只是"我的队伍更强了"，更是"我有科比了"。赛季任务线、挑战赛、周常活动构成了一个清晰的"日程表"，玩家每天有事可做，每周有小目标，每赛季有大目标。这种结构让"肝"变得有方向感，不容易产生虚无感。

我认为街球派对在留存设计上更成熟的核心原因在于：它把数值成长和情感认同绑定在一起了。你肝的不只是属性，你肝的是那个球星、那个皮肤、那个属于你的上场阵容。这种设计让流失成本更高，因为离开意味着放弃一个已经有情感投入的"身份"。

**3. 我的思考总结**

SPG 品类最核心的设计挑战，是如何在"运动竞技的真实感"和"游戏化的爽感"之间找到平衡点。端游可以偏向前者，手游必须偏向后者——但这不意味着手游就是简化版，街球派对证明了手游可以在情感设计和留存机制上做得更精细。

#### 二、开放世界

**开放世界品类分析 -- 以《对马岛之魂》《羊蹄山之魂》为例**

**1. 我为什么喜欢开放世界这个品类**

开放世界对我的吸引力，从来不是"地图有多大"或者"内容有多少"，而是一种很难用其他品类复制的体验：在一个陌生的世界里，迷路是正当的，绕路是有意义的，偶然发现一件小事可以让你停下来二十分钟。

这种体验在《对马岛之魂》里被我第一次真正感受到。游戏开场不久，我骑马穿过一片金色的芦苇荡，风吹过，苇草齐刷刷倒向一个方向——我本能地跟着风走，没有任何任务标记，最后找到了一座隐藏的神社。那一刻没有什么大奖励，但那种"我自己发现了这里"的满足感，是很多游戏用再多UI设计都给不了我的东西。从那之后，我开始认真思考：开放世界这个品类，它真正在卖的是什么？

**2. 游戏体验：一个值得慢下来的世界**

对马岛的体验节奏是我玩过的开放世界里最特别的之一——它鼓励你慢下来。

大多数开放世界游戏会用密密麻麻的地图图标告诉你"这里有事情要做"，探索变成了一份待办清单，玩家跟着标记走，世界本身反而成了通道。对马岛拒绝了这套逻辑。它用自然线索代替UI指引：金色芦苇随风倒向神社方向，狐狸出现时跟着走会发现神祠，远处的烟柱意味着有村庄需要救援。风向本身就是导航系统。这个设计带来的体验是：你不是在执行地图，你是在真正地旅行。我记得有一次支线，任务起点只是一具无名的尸体，没有标记、没有说明，我顺着线索一路追查，最后拼出了一个完整的悲剧故事——一个父亲为了保护女儿独自赴死。全程没有任何对话框告诉我"这里有个故事值得你关注"，是场景本身把我留住的。完成主线和大部分支线之后，我有一个很强烈的感受：对马岛的世界是有重量的。这种重量不来自体量，而来自密度——每一片废墟、每一具尸体、每一面刻着字的石碑，背后都有人在认真想过"这里发生过什么"。

**《羊蹄山之魂》**把这种体验推到了更深的地方。作为续作，它把故事从"对外抗争"转向了"向内审视"，叙事主题变得更加私人：关于身份认同，关于归属感，关于阿依努原住民文化与外来者之间的张力。我在做羊蹄山支线的时候数次停下来，不是因为卡关，而是因为某段对话让我需要缓一缓。这是开放世界叙事给我带来的体验上限——它让我觉得这个世界里有真实存在过的人。

**3. 文化氛围：沉浸感的来源**

对马岛对日本武士美学的还原是我认为它区别于同类游戏最核心的竞争力之一。它呈现的不只是武士的"打"，而是武士的"道"——荣誉、孤独、死亡与牺牲。

游戏的视觉语言极为克制：居合拔刀的慢动作、枯山水庭院、与自然深度融合的神社建筑、大片留白式的构图……它的美不是炫技式的，而是呼吸式的。我印象最深的一个画面是仁在雨中独自站在父亲墓前，没有音乐、没有旁白，只有雨声。那种孤独感用任何文字都说不清楚，但它就是渗进来了。

《羊蹄山之魂》在此基础上引入了阿依努原住民文化，这在游戏里极为罕见。阿依努的神话体系、自然崇拜、纹身图腾被认真地融进了世界，而不是作为异域装饰品。DLC在讲一个关于"谁才是这片土地真正主人"的更复杂的故事，这让它在文化呈现上拥有了超越主游戏的厚度。

**4. 横向对比：对马岛在品类里的独特位置**

我玩过一些同类开放世界作品，试着做一个简单的比较来说清楚对马岛的特殊性。《塞尔达：旷野之息》的探索驱动力是"我能做什么"——它的物理引擎和化学系统给了玩家极高的行动自由度，探索的快乐来自创造与实验。《荒野大镖客2》和对马岛更接近，都用环境细节和叙事密度驱动探索，但荒野大镖客更慢、更沉重，是一首挽歌；对马岛节奏更流畅，在沉浸感和爽感之间找到了一个更大众友好的平衡点。育碧系开放世界（刺客信条、看门狗）则走向了另一个极端——用极大的内容体量制造"值回票价"的感觉，但密度不够，探索的惊喜感被稀释了。

对马岛的独特之处在于它是一款用"减法"做出来的开放世界：去掉地图标记的强制依赖，去掉支线任务的流水线奖励逻辑，去掉视觉上的炫技冲动，留下的是情绪本身。在当前开放世界同质化严重的市场里，这种克制反而是最大的差异化。

**5. 思考与总结**

玩完对马岛系列之后，我对开放世界这个品类形成了一个核心判断：**开放世界的天花板不是技术参数，而是情感密度。**一个让玩家愿意慢下来、愿意绕路、愿意在一个无名武士的墓碑前停留三秒的世界，才是真正意义上的开放世界。对马岛之所以能做到这一点，是因为它的每一个设计决策——从导航系统到支线结构到视觉语言——都在服务同一个目标：让玩家在这个世界里产生真实的情感涟漪。这个判断让我开始重新审视"开放世界"这个品类的设计边界。它不一定要大，不一定要满，但它必须要让人觉得：这里有值得在意的东西，值得我多花一点时间。而对马岛做到了。

#### 三、简化版答案

**一、SPG 体育游戏——以《NBA 2K》系列与《全明星街球派对》为例**

我对篮球游戏的理解从 NBA 2K 开始。2K 的核心哲学是"拟真"——投篮有节奏条，运球有动量惯性，满足的是"我在打篮球"的幻想，门槛高，沉浸感强。但这套逻辑放到手游上行不通。手机屏幕、触控操作、碎片化时间，决定了不能让玩家适应高学习曲线。《全明星街球派对》的选择是聪明的：放弃拟真，追求爽感，满足的是"我在让球星打篮球"的幻想。两款游戏锚定了不同的用户心理：2K给的是"沉浸与掌控"，街球派对给的是"认同与表达"。

街球派对有一个我很欣赏的设计亮点：非明星球员也能凭借独特机制发挥关键作用。以我最喜欢的阿隆·戈登为例，他的天赋技能"争抢补篮"——被封盖时跳起争夺球权，可直接补进或传球——完美还原了他在现实中的身体型打法。一次对方超级球星的完美盖帽，被戈登就地补进，这一幕还原了篮球里真实存在的"以劣胜优"的戏剧性。每个球员都是一套独特的战术假设，而非数值排序的结果，这让游戏策略深度远超一般SPG手游。

SPG品类的核心挑战是在"竞技真实感"和"游戏化爽感"之间找到平衡。街球派对证明了手游可以在情感设计上做得更精细，而不只是简化版。

**二、开放世界——以《对马岛之魂》《羊蹄山之魂》为例**

开放世界对我的吸引力从来不是地图有多大，而是一种很难被复制的体验：迷路是正当的，绕路是有意义的，偶然发现一件小事可以让你停下来二十分钟。

《对马岛之魂》让我第一次真正感受到这件事。游戏开场，我骑马穿过金色芦苇荡，风吹来，苇草齐刷刷倒向一个方向，我本能地跟着走，没有任何任务标记，最后找到一座隐藏神社。没有大奖励，但"我自己发现了这里"的满足感，是再多UI设计都给不了的。对马岛用自然线索代替UI指引：芦苇随风指向神社，狐狸带路通往神祠，风向就是导航系统。体验的结果是：我不在执行地图，而是在真正旅行。

续作《羊蹄山之魂》引入了阿依努原住民文化，神话体系、自然崇拜、纹身图腾被认真融进世界，而非异域装饰品。我做支线时数次停下来，不是因为卡关，而是某段对话需要我缓一缓。这是开放世界叙事能给我的体验上限。

玩完这两部作品，我形成了一个核心判断：开放世界的天花板不是技术参数，而是情感密度。不一定要大，不一定要满，但必须让人觉得这里有值得在意的东西。对马岛和羊蹄山做到了。

### 问题 5 游戏产品拆解

#### **《全明星街球派对》游戏分析** 

**SPG品类 · 篮球手游 · 网易**

**1. 游戏定位与品类价值**

《全明星街球派对》是网易旗下以NBA授权球星为核心IP的3v3街头篮球手游。它并不试图做一款"手机上的2K"，而是选择了一条截然不同的路径：以街球文化为外壳，以球星情感为钩子，以爽感设计为驱动，面向中国移动端篮球爱好者构建了一个独特的产品生态。单局10分钟内完成，天然契合碎片化使用场景；NBA官方授权的球星IP则为产品提供了强力的情感入口和差异化壁垒。

**2. 核心玩法与手感设计**

街球派对的玩法设计核心是一个清晰的判断：手机端无法还原真实篮球的操控深度，那就做极致的"感觉上的篮球"。它的手感设计并不追求物理正确，而是追求每一个操作的即时反馈和情感满足——技能键大、反馈即时，进球特效夸张，3v3节奏快、失误容错高，不容易让玩家产生挫败感。

对比NBA 2K，2K追求的是拟真：节奏条、动量惯性、防守脚步判定，每一个细节都在模拟真实篮球的物理逻辑，给的是"我真的在打篮球"的沉浸体验。街球派对满足的则是另一种幻想——"我在用科比打篮球"。前者的门槛更高，流失也更快；后者的体验更浅，但情感锚点更稳。两款游戏各自锚定了不同的用户心理需求：2K给的是"沉浸与掌控"，街球派对给的是"认同与表达"。

**气势机制，让比赛拥有情绪叙事**，这是局内体验设计中最值得重点分析的部分。它的本质是一套将比赛进程转化为情绪波峰的动态反馈系统——连续得分、盖帽、抢断积累气势值，触发爆发后全队属性提升并伴随视觉特效强化，打开一个高光时刻窗口。

这个机制在体验层面产生了三个重要效果：第一，它给了落后方翻盘机会，哪怕分差存在，一波气势爆发就能重建比赛张力，防止单方面碾压带来的负体验流失；第二，它让普通玩家也能经历比赛高潮——一次关键抢断触发气势，技术一般的玩家同样能感受到英雄时刻，提升参与感；第三，它强化了团队协作的价值，气势由全队行为共同积累，鼓励3v3场景下的协同配合而非单人carry，贴合街球的文化气质。

气势机制的底层设计逻辑，是把"篮球比赛的戏剧性"系统化、可控化地植入游戏流程。它不是随机事件，而是玩家可以主动追求和触发的叙事工具，这正是它比单纯数值设计更高明的地方。

**3. 球员设计：现实特质的游戏化还原**

街球派对的球员设计有一个被低估的亮点：它不只是把NBA球星的名字和外貌搬进游戏，而是真正把球员在现实中的打球风格、身体特质和技术标签，转化成了独特的游戏机制。这让每个球员都有无可替代的战术价值，也让真正了解篮球的玩家产生强烈的"被懂得"的共鸣。更值得关注的是，这套设计打破了"明星球员必然更强"的线性逻辑。一些现实中并非超级巨星的角色，凭借独特机制在特定场景下完全可以压制超级明星。

以我最喜欢的角色阿隆·戈登为例。现实中的戈登以出色的运动能力和护框拼抢著称，而非得分或持球组织。游戏将这一特质精准提炼为他的天赋技能——"争抢补篮"：戈登触发"力大无穷"效果被封盖时，会跳起争夺球权，成功后可直接在空中将球补进篮筐或传球。这个机制的设计价值体现在三个层面：其一，现实还原度极高，戈登在NBA中正是以冲击篮框、争抢二次进攻机会著称，技能是其真实打法的高度抽象；其二，它赋予了逆转关键球的能力——被盖帽本是一次失败的进攻，但戈登能将其转化为二次得分机会，这种"失败中反击"的机制在关键时刻具有极强的心理震慑力；其三，这是一种非线性强度设计，戈登不需要在属性数值上压过超级明星，而是通过独特机制在特定场景中获得优势，让非顶级球星也拥有无可替代的战术地位。在实际对局中，对方超级球星一次看似完美的盖帽，结果被戈登凭借争抢补篮直接收入囊中——这一幕之所以令人印象深刻，正是因为它还原了篮球里真实存在的"以劣胜优"的戏剧性。戈登的案例揭示了街球派对在角色设计上的核心哲学：每个球员都是一套独特的战术假设，而不是数值高低的排序结果。这让游戏的策略深度远超一般SPG手游，也让玩家对"找到适合自己风格的球员"这件事产生了真正的动力。

**4. 球星收集系统与留存设计**

街球派对的收集系统是其商业模型与留存机制的核心。它的聪明之处在于：球星不只是数值容器，而是情感锚点。收集一个球星，意味着获得了一种身份认同，而不仅仅是属性提升。

游戏构建了一套多层级的球星体系：S+传奇球星赛季限定，稀有度极高且下架后不再复刻，制造明确的限定焦虑，配合专属入场动画和语音强化社交展示价值；S级明星球员作为核心竞技力来源，养成周期长，绑定中长期留存；A级普通球员门槛低，让新玩家快速感受到"拥有球星"的满足，降低早期流失风险。

在养成系统上，球星的成长路径被拆解为多个小目标：技能解锁、属性突破、专属装备升级、徽章系统……每一步都有可见的进度条和即时奖励，把"肝"这件枯燥的事包装成了有方向感的球星培养叙事。

留存节奏上，日任务维持日活、周挑战防止短期流失、赛季通行证绑定整季留存、限定球星触发付费冲刺，四层结构层层递进，让玩家在任何阶段都有清晰的目标感。

**5. 总结**

街球派对在手游赛道上做到了两件关键的事：第一，重新定义了"篮球游戏的核心乐趣"——不是拟真，而是幻想实现；第二，把情感认同编织进了数值成长，让玩家留下来不只是因为"还没打够"，而是因为"这是我的阿隆戈登"。气势机制赋予了比赛情绪起伏的叙事感，球员设计的现实还原让篮球迷产生深层共鸣，收集系统的分层结构保证了从新手到核心用户的完整留存路径。三者共同构成了一款在产品逻辑层面相当成熟的竞技手游。当然，数值膨胀过快、赛季重置对积累感的损耗，依然是SPG品类手游普遍面临的设计挑战，也是值得持续优化的方向。
