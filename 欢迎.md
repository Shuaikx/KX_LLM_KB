## 快速命令

```

```

## 目录

| 目录 | 用途 |
|------|------|
| `raw/` | 原始资料（你放入，Agent 只读） |
| `wiki/` | 编译后知识（Agent 维护） |
| `output/` | 报告与可视化输出 |
